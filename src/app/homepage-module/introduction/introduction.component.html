Genetically is a Typescript library made for genetical algorithm. The goal of this library is to provide powerful tools to build Genetic algorithm.

This library is inspired by Genetic Algorithm, the book by " Algorithmes génétiques exploration, optimisation et apprentissage automatique, by David-E Goldberg ".

With a few examples, we will discover the basics of genetic algorithm using Genetically. We will also discover how to use the library for basic creation of Genetical Algorithm.

The project is hosted publicly on github. Find informations about install and usage on on the project Readme.
